## Deployment Steps

### 1. Commit & Push to Git

```bash
git add .
git commit -m "prepare for Render deployment"
git push origin main
```

### 2. Deploy on Render

**Quick Deploy (Recommended):**

1. Go to https://dashboard.render.com
2. Click **New** → **Blueprint**
3. Connect your GitHub/GitLab repository
4. Render will auto-detect `render.yaml` and deploy

**Manual Deploy Alternative:**

1. Go to https://dashboard.render.com
2. Click **New** → **Web Service**
3. Configure as shown in DEPLOYMENT.md

### 3. Access Your App

- **Frontend:** `https://<your-service-name>.onrender.com`
- **API Docs:** `https://<your-service-name>.onrender.com/api-docs`
- **Demo Account:** admin@finflow.com / admin123

## Important Notes

⚠️ **SQLite Limitation:** Render uses ephemeral storage. Database resets on redeploy.

- For production, consider upgrading to PostgreSQL (see DEPLOYMENT.md)
